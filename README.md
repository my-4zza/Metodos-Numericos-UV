Este repositorio está dedicado a la experiencia educativa de Métodos Numéricos. Contiene el material en código de las actividades y reportes solicitados, enfocado en la implementación de algoritmos matemáticos en Python.

## Códigos Incluidos

El repositorio contiene 5 programas fundamentales para encontrar raíces de ecuaciones (métodos cerrados y abiertos). Cada uno solicita la función y los parámetros al usuario, calcula la raíz iterativamente y genera dos gráficas (el comportamiento de la función y la evolución del error absoluto).

1. **Método de Bisección** 2. **Método de Falsa Posición** 3. **Método del Punto Fijo** 4. **Método de Newton-Raphson** 5. **Método de la Secante**
## Requisitos de Instalación

Para ejecutar los programas correctamente, es necesario tener Python instalado junto con las siguientes librerías:

- `sympy` (Para la evaluación simbólica de funciones y derivadas)
- `numpy` (Para el manejo de arreglos de datos)
- `matplotlib` (Para la generación de gráficas de funciones y errores)

Se puede instalarlas usando pip:
```bash
pip install sympy numpy matplotlib
